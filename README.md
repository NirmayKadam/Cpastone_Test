# Quant Terminal Monorepo

Phase 1 bootstrap for a quantitative derivatives trading terminal.

## Services
- `apps/api-gateway`
- `apps/market-data-service`
- `apps/quant-engine`
- `apps/ml-service`
- `apps/worker-orchestrator`
- `apps/frontend`

## Quick start
1. Copy `.env.example` to `.env`.
2. Start the stack with `docker compose up --build`.
3. Validate health endpoints on exposed services.
